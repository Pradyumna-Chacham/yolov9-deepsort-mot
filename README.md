# Multi-Object Tracking in Cluttered Scenes

A modular video understanding pipeline using YOLOv9 + DeepSORT for multi-object tracking, event extraction for scene reasoning, and BLIP-based captioning.

## Planned Features

- YOLOv9 detection
- DeepSORT tracking
- MOT17 benchmarking
- Rule-based event extraction
- Template captions
- Optional BLIP captioning
- Gradio demo
- Hugging Face Spaces deployment
